⚙️ Instalación

Clona este repositorio:

git clone https://github.com/tu-usuario/tu-repositorio.git

Ingresa al proyecto:

cd tu-repositorio

Instala las dependencias:

npm install

▶️ Ejecutar Cypress

Abrir Cypress en modo interactivo:

npx cypress open

Ejecutar pruebas en consola:

npx cypress run
